## 🚀 How to Get Started

### Step 1: Install (2 min)

```bash
npm install --legacy-peer-deps
```

### Step 2: Configure (2 min)

```bash
cp .env.example .env.local
# Add MongoDB URL
```

### Step 3: Setup Database (2 min)

```bash
npm run db:push
npm run db:seed
```

### Step 4: Run (1 min)

```bash
npm run dev
```

### Step 5: Test (5 min)

- Visit http://localhost:3000
- Login as admin@store.com / Admin@123
- Test all features

**Total Time: ~12 minutes**
